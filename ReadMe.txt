Описание файлов в домашнем задании:

- CommandLineScript - скрипт для запуска из командной строки.

- ModuleScript - скрипт для создания декоратора, позволяет импортировать декоратор в свой код как отельный модуль.

- ContextModuleScript - скрипт для использования класса в контекстном менеджере.

Описание работы каждого скрипта с примерами - внутри скрипта.
